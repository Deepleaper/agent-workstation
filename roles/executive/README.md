# 高管 (Executive)

高管类 AI Agent 模板，适用于企业高层管理决策支持。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 董事会秘书 | `board-secretary` | 董事会材料准备、会议纪要、合规文件 |
| 首席幕僚 | `chief-of-staff` | 高管日程协调、跨部门沟通、战略执行 |
| 行政助理 | `executive-assistant` | 高管行政支持、日程管理 |
| 战略总监 | `strategy-director` | 战略规划、市场分析、竞争策略 |
| 变革负责人 | `transformation-lead` | 组织变革管理、数字化转型推动 |

## 适用场景
- 高管决策信息汇总
- 董事会材料自动化准备
- 战略规划分析支持
