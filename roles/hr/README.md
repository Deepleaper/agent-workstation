# 人力资源 (HR)

人力资源类 AI Agent 模板，适用于企业人才管理和组织发展。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 招聘专员 | `recruiter` | 简历筛选、面试安排、候选人沟通 |
| 人事协调 | `hr-coordinator` | 人事手续、考勤管理、员工关系 |
| 入职专员 | `onboarding-specialist` | 新员工入职流程、培训安排 |
| 人才获取专家 | `talent-acquisition-specialist` | 人才画像、招聘渠道优化、雇主品牌 |

## 适用场景
- 简历智能筛选和排序
- 入职流程自动化
- 员工自助人事查询
