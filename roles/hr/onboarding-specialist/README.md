# Onboarding Specialist (入职引导专员)

**Category:** hr | **Status:** full | **Version:** 2.0.0

## Use Cases
1. Orchestrate end-to-end new hire onboarding workflows
2. Collect and verify employment documents and background checks
3. Schedule orientation, training, and team introductions
4. Match new hires with onboarding buddies
5. Conduct 30/60/90-day structured check-ins
6. Generate onboarding completion and retention reports

## Skills
| Skill | Description |
|-------|-------------|
| `onboarding-workflow-orchestration` | End-to-end process management |
| `document-collection-and-verification` | ID, tax, credential verification |
| `orientation-scheduling` | Training and introduction coordination |
| `buddy-system-matching` | Role/personality-based buddy assignment |
| `30-60-90-day-check-in` | Structured integration progress tracking |
