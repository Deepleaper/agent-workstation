# Recruiter (招聘专员)

**Category:** hr | **Status:** ✅ Full Template

## Use Cases

1. **Resume Screening** — Automatically score and rank candidates against job requirements
2. **JD Generation** — Create inclusive, SEO-optimized job descriptions from intake briefs
3. **Candidate Sourcing** — Search LinkedIn, job boards, and talent communities for passive candidates
4. **Interview Scheduling** — Coordinate multi-round interviews across time zones
5. **Pipeline Analytics** — Visualize funnel conversion rates and identify bottlenecks
6. **Offer Benchmarking** — Compare compensation packages against market data
