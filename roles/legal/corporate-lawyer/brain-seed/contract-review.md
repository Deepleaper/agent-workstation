# 合同审查清单

## 一、基本信息核查

- [ ] 合同双方主体信息是否准确（公司名称、统一社会信用代码）
- [ ] 签约代表是否有授权（法定代表人或授权委托书）
- [ ] 合同签订日期和生效日期是否明确
- [ ] 合同编号是否按规范编制

## 二、核心条款审查

### 2.1 标的条款
- [ ] 标的物/服务描述是否清晰、具体、无歧义
- [ ] 数量、质量标准是否明确
- [ ] 技术规格或验收标准是否详细

### 2.2 价格与支付条款
- [ ] 总价及单价是否明确
- [ ] 付款方式（预付/分期/到付）是否合理
- [ ] 付款条件和时间节点是否清楚
- [ ] 发票类型和开具时间是否约定
- [ ] 是否有价格调整机制

### 2.3 履行期限与方式
- [ ] 交付时间/服务期限是否明确
- [ ] 交付地点和方式是否约定
- [ ] 分批交付的各阶段时间表是否合理

### 2.4 知识产权条款
- [ ] IP 归属是否明确（特别是软件开发、设计类合同）
- [ ] 第三方知识产权使用授权是否合规
- [ ] 是否有竞业限制相关条款

## 三、风险条款审查

### 3.1 违约责任
- [ ] 违约情形是否明确列举
- [ ] 违约金比例是否合理（通常不超过合同总额30%）
- [ ] 损害赔偿范围是否界定（直接损失/间接损失）

### 3.2 保密条款
- [ ] 保密信息范围是否清晰
- [ ] 保密期限是否合理（一般2-5年）
- [ ] 违反保密义务的责任是否明确

### 3.3 不可抗力
- [ ] 不可抗力的定义和范围是否合理
- [ ] 通知义务和证明要求是否明确
- [ ] 不可抗力持续期间的处理方式

### 3.4 争议解决
- [ ] 争议解决方式（协商→调解→仲裁/诉讼）
- [ ] 管辖法院/仲裁机构的选择是否有利
- [ ] 适用法律是否明确（涉外合同特别注意）

## 四、特殊条款审查

- [ ] 合同变更和解除条件是否合理
- [ ] 通知条款（送达地址、方式）是否完整
- [ ] 合同附件是否齐全且与正文一致
- [ ] 是否有陷阱条款（自动续约、单方变更权等）
- [ ] 担保/保证金条款是否合理

## 五、审查结论模板

```
审查意见：
1. 风险等级：高/中/低
2. 主要风险点：
   - ...
3. 修改建议：
   - 第X条建议修改为...
4. 结论：建议签署 / 修改后签署 / 不建议签署
```
