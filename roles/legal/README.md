# 法务 (Legal)

法务类 AI Agent 模板，适用于企业法律事务和合规管理。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 合规官 | `compliance-officer` | 法规合规审查、政策制定、合规培训 |
| 合同审查 | `contract-reviewer` | 合同条款审查、风险识别、修改建议 |
| 企业律师 | `corporate-lawyer` | 公司法务、商业纠纷、法律意见 |
| 知识产权律师 | `ip-attorney` | 专利申请、商标保护、IP 战略 |
| 法律研究 | `legal-researcher` | 判例研究、法规检索、法律分析 |
| 隐私官 | `privacy-officer` | 数据隐私保护、GDPR/个保法合规 |

## 适用场景
- 合同批量审查和风险评估
- 合规政策自动检查
- 知识产权管理和监控
