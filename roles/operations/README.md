# 运营管理 (Operations)

运营管理类 AI Agent 模板，适用于企业供应链和运营效率优化。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 产能规划 | `capacity-planner` | 产能预测、资源调度、瓶颈分析 |
| 车队管理 | `fleet-manager` | 车辆调度、维护计划、路线优化 |
| 库存分析师 | `inventory-analyst` | 库存数据分析、补货策略优化 |
| 库存经理 | `inventory-manager` | 库存管理、出入库、盘点 |
| 物流协调 | `logistics-coordinator` | 运输安排、物流跟踪、配送优化 |
| 运营分析师 | `operations-analyst` | 运营数据分析、流程优化建议 |
| 流程改进专家 | `process-improvement-specialist` | 流程梳理、精益改善、效率提升 |
| 采购专员 | `procurement-specialist` | 供应商筛选、询价比价、采购执行 |
| 质检员 | `quality-inspector` | 质量检查、缺陷记录、质量报告 |
| 供应链经理 | `supply-chain-manager` | 供应链全流程管理和优化 |
| 供应商管理 | `vendor-manager` | 供应商评估、关系维护、绩效管理 |
| 仓库经理 | `warehouse-manager` | 仓储管理、布局优化、人员调度 |

## 适用场景
- 供应链全链条智能化
- 库存和物流自动化管理
- 采购和质量流程标准化
