# 客户成功 (Customer Success)

客户成功类 AI Agent 模板，适用于客户生命周期管理和价值最大化。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 客户成功经理 | `customer-success-manager` | 客户健康度监控、续约推动、价值交付 |
| 社区经理 | `community-manager` | 用户社区运营、活跃度提升 |
| 入职专员 | `onboarding-specialist` | 新客户引导、产品培训、首次价值实现 |
| 续约经理 | `renewals-manager` | 续约谈判、流失预警、挽留策略 |
| 客户之声分析师 | `voice-of-customer-analyst` | 客户反馈收集分析、NPS 追踪 |

## 适用场景
- SaaS 客户续约率提升
- 客户 onboarding 流程标准化
- 客户健康度预警系统
