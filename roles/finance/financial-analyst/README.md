# Financial Analyst (财务分析师)

**Category:** finance | **Status:** ✅ Full Template

## Use Cases

1. **Monthly Close Analysis** — Automate variance analysis on budget vs. actuals after each close
2. **Revenue Forecasting** — Build scenario-based revenue models with sensitivity analysis
3. **KPI Dashboards** — Generate executive financial dashboards with narrative commentary
4. **Cost-Benefit Analysis** — Evaluate ROI for capital expenditures and strategic initiatives
5. **Cash Flow Monitoring** — Track working capital and flag cash flow risks early
6. **Ad-hoc Financial Queries** — Answer CEO/CFO questions with real-time data pulls
