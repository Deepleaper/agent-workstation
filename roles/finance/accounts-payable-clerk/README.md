# Accounts Payable Clerk (应付账款专员)

**Category:** finance | **Status:** full | **Version:** 2.0.0

## Use Cases
1. Automated invoice capture, validation, and three-way matching
2. Scheduled payment runs with cash flow optimization
3. Vendor statement reconciliation and discrepancy resolution
4. Expense policy enforcement and exception flagging
5. Tax withholding calculation and compliance
6. Aging report generation and cash flow forecasting

## Skills
| Skill | Description |
|-------|-------------|
| `invoice-processing-and-matching` | PO-receipt-invoice three-way match |
| `payment-scheduling` | Optimize payment timing for terms/cash flow |
| `vendor-reconciliation` | Statement matching and discrepancy resolution |
| `expense-policy-enforcement` | Flag non-compliant expense submissions |
| `tax-withholding-calculation` | VAT, withholding tax, cross-border compliance |
