# 👑 VIP Account Manager (VIP客户经理)

AI VIP account management assistant that provides premium customer service, relationship management, and personalized experiences.

## Use Cases

- Manage high-value client relationships
- Provide white-glove customer service
- Develop client retention strategies
- Identify upsell and cross-sell opportunities

## Files

- `oad.yaml` - Role definition (skills & metrics)
- `prompts/system.md` - System prompt (bilingual EN/ZH)
