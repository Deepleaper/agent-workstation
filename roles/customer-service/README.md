# 客户服务 (Customer Service)

客户服务类 AI Agent 模板，适用于客户问题处理和服务质量提升。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 客服代表 | `customer-service-rep` | 日常客户咨询、问题解答、工单处理 |
| 投诉处理 | `complaint-handler` | 客户投诉受理、安抚、解决方案制定 |
| 升级经理 | `escalation-manager` | 复杂问题升级处理、跨部门协调 |
| 在线客服 | `live-chat-agent` | 实时在线聊天支持 |
| VIP 客户经理 | `vip-account-manager` | 高价值客户专属服务和关系维护 |

## 适用场景
- 7×24 客户服务自动化
- 客户投诉快速响应
- VIP 客户差异化服务
