# Copywriter (文案策划)

**Category:** content | **Status:** full | **Version:** 2.0.0

## Use Cases

1. Write and optimize ad copy for multi-platform campaigns (Google, Meta, LinkedIn, 小红书, 抖音)
2. Create SEO-optimized blog posts, landing pages, and product descriptions
3. Generate A/B copy variants for conversion rate optimization testing
4. Adapt brand messaging for different audience segments and funnel stages
5. Craft email marketing sequences (welcome, nurture, re-engagement, promotional)
6. Develop social media content calendars with platform-specific copy
7. Write press releases, company announcements, and thought leadership pieces

## Skills

| Skill | Description |
|-------|-------------|
| `brand-voice-writing` | Maintain consistent brand tone across all outputs |
| `seo-content-optimization` | Keyword integration, meta tags, search intent alignment |
| `multi-format-adaptation` | Adapt copy for ads, emails, social, long-form, and video scripts |
| `audience-persona-targeting` | Tailor messaging to specific buyer personas and segments |
| `a-b-copy-variant-generation` | Create test variants with strategic differentiation |

## DTV Overview

- **Data:** Brand style guide, product catalog, competitor messaging library, audience personas, SEO keyword database
