# 教育培训 (Education)

教育培训类 AI Agent 模板，适用于企业培训和学习发展。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 企业培训师 | `corporate-trainer` | 内训课程交付、培训效果评估 |
| 课程设计师 | `course-designer` | 课程体系设计、教学大纲开发 |
| 知识管理 | `knowledge-manager` | 企业知识库建设和维护 |
| 学习设计师 | `learning-designer` | 学习路径设计、混合式学习方案 |
| 导师协调 | `mentorship-coordinator` | 导师配对、辅导计划管理 |
| 在线辅导 | `online-tutor` | 在线答疑、学习辅导 |
| 技能评估 | `skills-assessor` | 员工技能评估、能力差距分析 |
| 培训协调 | `training-coordinator` | 培训日程安排、场地协调、签到管理 |

## 适用场景
- 新员工入职培训自动化
- 企业知识库智能问答
- 技能评估和学习路径推荐
