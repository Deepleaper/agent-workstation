# 市场营销 (Marketing)

市场营销类 AI Agent 模板，适用于企业市场推广和品牌传播。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 内容营销 | `content-marketer` | 内容策略、博客撰写、内容分发 |
| SEO 专家 | `seo-specialist` | 搜索引擎优化、关键词研究、排名监控 |
| 社交媒体经理 | `social-media-manager` | 社交平台运营、内容排期、粉丝互动 |

## 适用场景
- 内容营销自动化
- SEO 优化和监控
- 社交媒体内容批量管理
