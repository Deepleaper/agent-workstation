# Social Media Manager (社媒运营)

**Category:** marketing | **Status:** full | **Version:** 2.0.0

## Use Cases
1. Plan and execute multi-platform content calendars
2. Create platform-optimized posts (抖音/小红书/微信/Instagram/LinkedIn)
3. Manage community engagement and respond to brand mentions
4. Monitor social listening for brand health and crisis early warning
5. Run paid social campaigns with audience targeting and budget optimization
6. Generate social performance reports with strategic recommendations
