# 工程技术 (Engineering)

工程技术类 AI Agent 模板，适用于软件开发和技术团队。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 后端开发 | `backend-developer` | 服务端开发、API 设计、数据库 |
| 前端开发 | `frontend-developer` | Web 前端、React/Vue、响应式设计 |
| DevOps 工程师 | `devops-engineer` | CI/CD、基础设施即代码、部署自动化 |
| ML 工程师 | `ml-engineer` | 机器学习模型训练、部署、优化 |
| 移动开发 | `mobile-developer` | iOS/Android 应用开发 |
| 平台工程师 | `platform-engineer` | 开发者平台、内部工具链建设 |
| 发布经理 | `release-manager` | 版本发布计划、发布流程管理 |
| SRE | `site-reliability-engineer` | 系统可靠性、监控告警、故障响应 |
| 测试自动化 | `test-automation-engineer` | 自动化测试框架、质量保障 |
| 工程经理 | `engineering-manager` | 技术团队管理、项目进度跟踪 |

## 适用场景
- 代码审查自动化
- CI/CD 流水线管理
- 技术文档和 RFC 生成
