# 设计 (Design)

设计类 AI Agent 模板，适用于企业品牌和用户体验设计。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| UI 设计师 | `ui-designer` | 界面设计、组件设计、交互规范 |
| 品牌设计师 | `brand-designer` | 品牌视觉、VI 设计、品牌指南 |
| 动效设计师 | `motion-designer` | 动画效果、微交互、视频动效 |
| 设计系统负责人 | `design-system-lead` | 设计系统建设、组件库维护 |
| UX 文案 | `ux-writer` | 界面文案、微文案、用户引导文案 |

## 适用场景
- 设计资产批量生成
- 品牌一致性管理
- 设计系统维护和扩展
