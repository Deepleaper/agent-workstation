# 产品 (Product)

产品类 AI Agent 模板，适用于产品管理和用户研究。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 产品经理 | `product-manager` | 需求管理、产品规划、版本迭代 |
| 产品分析师 | `product-analyst` | 产品数据分析、漏斗分析、A/B 测试 |
| 产品负责人 | `product-owner` | 产品待办列表管理、迭代规划 |
| 增长产品经理 | `growth-product-manager` | 增长实验、用户获取、留存优化 |
| 技术产品经理 | `technical-product-manager` | 技术方案评审、API 产品管理 |
| UX 研究员 | `ux-researcher` | 用户研究、可用性测试、用户访谈 |

## 适用场景
- 产品需求整理和优先级排序
- 用户行为数据分析
- 竞品分析和市场调研
