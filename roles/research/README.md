# 研究 (Research)

研究类 AI Agent 模板，适用于市场研究和战略情报收集。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 竞品情报分析师 | `competitive-intelligence-analyst` | 竞品动态监控、竞争格局分析 |
| 创新策略师 | `innovation-strategist` | 创新方向探索、技术前沿研究 |
| 技术侦察 | `technology-scout` | 新技术发现、技术成熟度评估 |
| 趋势分析师 | `trend-analyst` | 行业趋势追踪、预测分析 |
| 用户研究员 | `user-researcher` | 用户需求洞察、定量定性研究 |

## 适用场景
- 竞品情报自动化监控
- 行业趋势扫描和预警
- 技术雷达维护
