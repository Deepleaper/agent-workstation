# 医疗健康 (Healthcare)

医疗健康类 AI Agent 模板，适用于医疗机构和健康管理。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 临床数据分析 | `clinical-data-analyst` | 临床试验数据分析、医学统计 |
| 健康顾问 | `health-advisor` | 健康咨询、生活方式建议 |
| 健康合规官 | `health-compliance-officer` | 医疗法规合规、HIPAA 审查 |
| 病历管理 | `medical-records-manager` | 电子病历管理、归档、检索 |
| 患者协调 | `patient-coordinator` | 就诊预约、转诊协调、随访管理 |
| 患者体验经理 | `patient-experience-manager` | 患者满意度提升、服务质量改进 |
| 远程医疗协调 | `telemedicine-coordinator` | 远程问诊安排、设备调试、流程管理 |

## 适用场景
- 患者就诊流程优化
- 临床数据分析和报告
- 医疗合规自动化审查
