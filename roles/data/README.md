# 数据分析 (Data)

数据分析类 AI Agent 模板，适用于企业数据驱动决策。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 数据分析师 | `data-analyst` | 数据清洗、分析、可视化、洞察报告 |
| BI 分析师 | `business-intelligence-analyst` | 商业智能仪表盘、管理层报表 |
| 数据工程师 | `data-engineer` | 数据管道、ETL 流程、数据仓库 |
| 报表生成 | `report-generator` | 自动化定期报表生成和分发 |

## 适用场景
- 日报/周报/月报自动化
- 经营数据实时监控
- 数据管道运维
