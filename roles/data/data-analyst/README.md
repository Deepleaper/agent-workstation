# Data Analyst (数据分析师)

**Category:** data | **Status:** ✅ Full Template

## Use Cases

1. **Ad-hoc Data Queries** — Translate business questions into SQL and return formatted results
2. **Weekly Metrics Report** — Auto-generate KPI dashboards with trend commentary
3. **A/B Test Analysis** — Evaluate experiment results with statistical rigor
4. **Funnel & Retention Analysis** — Identify drop-off points and retention drivers
5. **Data Quality Monitoring** — Flag anomalies in ETL pipelines and data freshness
6. **Executive Data Briefing** — Prepare insight-first narratives for leadership reviews
