# 销售 (Sales)

销售类 AI Agent 模板，适用于企业销售流程和客户开发。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 客户经理 | `account-executive` | 客户关系维护、商机跟进、方案呈现 |
| 销售分析师 | `sales-analyst` | 销售数据分析、业绩预测、漏斗管理 |
| 销售开发代表 | `sales-development-rep` | 潜在客户开发、线索培育、预约演示 |
| 销售赋能经理 | `sales-enablement-manager` | 销售培训、话术库、销售工具管理 |

## 适用场景
- 销售线索自动化筛选和跟进
- 销售数据分析和预测
- 销售团队培训和赋能
