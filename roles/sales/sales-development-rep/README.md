# Sales Development Representative (销售拓展代表)

**Category:** sales | **Status:** ✅ Full Template

## Use Cases

1. **Lead Qualification** — Score and qualify inbound/outbound leads using BANT/MEDDIC
2. **Outbound Sequences** — Generate and execute personalized multi-touch email cadences
3. **Account Research** — Enrich target accounts with firmographic and technographic data
4. **Meeting Booking** — Schedule qualified demos/calls and hand off to AEs with context
5. **Pipeline Hygiene** — Keep CRM data clean with automated status updates and activity logging
6. **Cadence Optimization** — Analyze open/reply rates and recommend messaging improvements
