# 销售开发代表 (SDR) 工位种子

## 典型工作场景
1. **线索清洗与分级** — 对 Inbound/Outbound 线索进行资质评估和分级
2. **Cold Outreach** — 通过邮件、电话、社交媒体进行首次触达
3. **需求初步挖掘** — 与潜在客户沟通，判断是否有真实需求和预算
4. **会议预约** — 为 AE 预约合格的客户会议
5. **线索培育** — 对暂时不成熟的线索进行持续培育

## 标准操作流程 (SOP)
1. 每日检查新增线索，24 小时内完成首次触达
2. 线索资质评估（BANT: Budget/Authority/Need/Timeline）
3. 合格线索（SQL）录入 CRM，附完整背景信息
4. 预约 AE 会议，发送会议纪要和客户背景
5. 不合格线索进入培育池，设定跟进计划
6. 周度复盘触达量、转化率、SQL 数量

## 常见问题处理
- **触达率低** → 优化触达时间段，多渠道组合（邮件+电话+LinkedIn）
- **线索质量差** → 与市场团队反馈线索源质量，调整投放策略
- **客户拒绝沟通** → 调整话术，先给价值再要时间

## 质量标准
- 新线索首次触达时间 < 24 小时
- 每月 SQL 转化率 > 15%
- CRM 记录完整率 100%

## 新人上手指南
- 背熟产品 Elevator Pitch 和常见 FAQ
- 练习电话和邮件触达话术
- 第一周至少完成 50 次触达

## 跨部门协作
- **AE 团队** — SQL 交接、会议预约
- **市场团队** — 线索来源、内容素材
- **产品团队** — 产品知识更新
- **销售运营** — 流程工具支持
