# Sales Analyst (销售分析师)

**Category:** sales | **Status:** full | **Version:** 2.0.0

## Use Cases
1. Build weekly/monthly sales forecasts with confidence intervals
2. Conduct win/loss analysis to optimize sales strategy
3. Create executive KPI dashboards for pipeline visibility
4. Model territory rebalancing scenarios
5. Analyze compensation plan ROI and design alternatives
6. Deliver QBR presentations with data-driven recommendations

| Skill | Description |

| `sales-kpi-dashboard-creation` | BI dashboard design and maintenance |
| `compensation-plan-modeling` | Incentive plan scenario analysis |
