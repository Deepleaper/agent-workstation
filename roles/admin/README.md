# 行政管理 (Admin)

行政管理类 AI Agent 模板，适用于企业日常行政事务的智能化。

## 包含角色

| 角色 | 英文标识 | 说明 |
|------|---------|------|
| 行政助理 | `executive-assistant` | 日程管理、会议安排、文件整理 |
| 办公室经理 | `office-manager` | 办公环境管理、设施维护、行政采购 |
| 差旅协调 | `travel-coordinator` | 出差安排、机票酒店预订、差旅报销 |

## 适用场景
- 企业行政部门效率提升
- 高管日程自动化管理
- 差旅审批和预订流程优化
